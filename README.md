# Astrix Rover Website

Official website of the Astrix Rover Team, showcasing our mission, projects, achievements, team members, and innovations in robotics and autonomous systems.

## About the Project

The Astrix Rover Website serves as the digital platform for our robotics team. It provides visitors with information about our rover development journey, ongoing projects, technical expertise, team members, and events.

The website is designed with a modern and responsive interface to ensure a seamless experience across desktop, tablet, and mobile devices.

## Features

- Responsive Design
- Interactive User Interface
- Smooth Navigation
- Team Showcase
- Project Portfolio
- Contact Section
- Mobile-Friendly Layout
- Optimized Performance

## Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Google Fonts
